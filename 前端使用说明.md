# 前端使用说明

## ⚠️ 重要提示

**老的前端（Flask 模板系统）已不再使用！**

`dispatch_center/templates/index.html` 和 `dispatch_center/static/js/app.js` 等文件为旧版前端代码，现已废弃，请勿使用。

---

## ✅ 新前端（Vue3）

项目现已迁移到 **Vue3 + TypeScript + Vite** 技术栈。

### 目录结构

```
vue-frontend/          # Vue 前端项目目录
├── src/
│   ├── components/    # Vue 组件
│   ├── stores/        # Pinia 状态管理
│   ├── api/           # API 接口
│   └── types/         # TypeScript 类型定义
├── package.json
└── vite.config.ts
```

### 开发环境启动

1. **启动 Flask 后端**（API 服务）
   ```bash
   cd dispatch_center
   python app.py
   ```
   后端运行在 http://127.0.0.1:5001

2. **启动 Vue 前端**（开发服务器）
   ```bash
   cd vue-frontend
   npm install    # 首次运行需要安装依赖
   npm run dev    # 启动开发服务器
   ```
   前端运行在 http://localhost:5173

3. **访问系统**
   打开浏览器访问 http://localhost:5173

### 生产环境部署

```bash
cd vue-frontend
npm run build
```

构建后的文件会输出到 `dispatch_center/static/vue-dist/` 目录。

---

## 🔧 注意事项

1. **不要再访问 `http://127.0.0.1:5001`** - 这是旧版前端地址，已废弃
2. **Vue 前端代理配置**：开发时所有 `/api` 请求会自动代理到 Flask 后端 (5001 端口)
3. **同步异常提示**：如果页面显示同步异常，请检查：
   - Flask 后端是否已启动
   - MySQL 数据库 (192.168.1.88) 是否可连接

---
